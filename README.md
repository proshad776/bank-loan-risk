
# 🏦 Bank Loan Risk & Customer Analytics Dashboard

## 📌 Overview

The Bank Loan Risk & Customer Analytics Dashboard is an interactive Power BI project designed to analyze loan applications, customer demographics, financial risk, and lending performance. The dashboard transforms raw banking data into meaningful insights, helping stakeholders understand loan approval trends, customer profiles, and risk indicators for informed decision-making.

---

## 🎯 Project Objectives

- Analyze loan application trends.
- Evaluate customer demographics.
- Monitor loan approval and rejection rates.
- Identify high-risk customer segments.
- Track key financial KPIs.
- Support data-driven lending decisions.

---

## 📊 Dashboard Features

- Loan Application Overview
- Loan Approval vs. Rejection Analysis
- Customer Demographics
- Income Analysis
- Loan Purpose Distribution
- Risk Assessment
- Interactive Filters & Slicers
- KPI Cards

## 📂 Dataset

The project uses a banking dataset containing customer demographics, loan information, income, employment details, loan status, credit-related attributes, and financial indicators.

---

## 📈 Key Performance Indicators (KPIs)

- Total Loan Applications
- Total Loan Amount
- Approved Loans
- Rejected Loans
- Approval Rate
- Average Loan Amount
- Customer Count
- Average Income
- Risk Distribution

---

## 🔍 Business Insights

- Identified customer groups with higher loan approval rates.
- Analyzed demographic trends affecting loan decisions.
- Compared loan performance across different categories.
- Visualized customer income and loan amount relationships.
- Highlighted financial risk indicators using interactive dashboards.
